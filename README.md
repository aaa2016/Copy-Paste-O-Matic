## Purpose<a name="purpose"></a>

This tool allows a user to quickly paste snippets of text by clicking the corresponding button.

The text is appended to the text area on the right.

Click 'COPY TEXT' to copy the contents of the text area to your clipboard, for pasting elsewhere.

The sections, buttons, and text to be inserted from each button are found in the config.js file - this file must be saved in the same directory as the .html file of this tool.

Each snippet of text can be copied as many times as needed.

Any text that is manually inserted or edited in the text area will also be copied to the clipboard.

The user can select the option of a single or double carriage return (new line) between copied snippets of text.
